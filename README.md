⚙️ Setup Instructions
1️⃣ Clone Repository
git clone <your-repo-url>
cd todo-app

2️⃣ Backend Setup
Navigate to backend
cd backend

Install dependencies
npm install

Create .env file
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

Start backend server
npm run dev


Backend will run at:

http://localhost:5000

3️⃣ Frontend Setup
Navigate to frontend
cd frontend

Install dependencies
npm install

Start frontend
npm run dev


Frontend will run at:

http://localhost:5173
